1. Nome do Projeto: Controle Financeiro VOF

2. Objetivo:
Ajudar pessoas a ter controle completo das suas finanças pessoais, registrando receitas, despesas e acompanhando o saldo mensal de forma simples e organizada.

3. Tecnologias:
Utilize PHP e banco de dados MySQL.

4. Design, Cores e Layout:
- Interface limpa, minimalista e profissional.
- As cores devem ser baseadas na paleta de cores tons de azul, verde e cinza claro.
- Deve ser responsivo para qualquer dispositivo e tamanho de tela.
- Focado na melhor experiência de uso possível (UX).
- Layout moderno com sidebar à esquerda.
- Use a fonte "Poppins".
- Sidebar com logo no topo e menu de navegação.
- Separe o arquivo menu.php que implementa o menu sidebar.

Botões de ação:
- Primário em azul
- Secundários em branco

5. Requisitos importantes
- O sistema deve ter login (sessões PHP nativas).
- Cada usuário deve ver apenas os seus próprios dados (arquitetura multi-tenancy).
- Código claro, fácil de entender e evoluir.
- Projeto preparado para evolução futura (integração com bancos, notificações, WhatsApp, etc.).
- Não permitir duas contas de usuário com o mesmo e-mail.
- Máscara de telefone: Formato brasileiro (XX) XXXXX-XXXX.
- As senhas dos usuários devem ter no mínimo 6 caracteres.
- Formato de moeda: R$ com separador de milhar.
- Formato de data: dd/mm/yyyy com calendário.

6. Funcionalidades da aplicação para a área deslogada (sem login)
- Tela de Boas-vindas
- Página inicial apresentando o sistema e os benefícios de manter um controle financeiro pessoal organizado.
